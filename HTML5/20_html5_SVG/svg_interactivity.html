SVG Interactivity:scalable vector graphics

      Difference between raster/bitmapped image and SVG
      Raster/bitmapped image: composed of a series of dots, each dot or bit is mapped to a color.
      SVG: vector image is just a description of basic shapes, lines, text and device independent..
      
      Difference canvas and SVG
      SVG: declarative, based on xml, vector, every element is an javascript object
      Canvas: based on javascript, raster
