In this article I'll describe the World Wide Web from a high perspective, focusing on how a web page is request and delivered from a web server to a web browser.
Then, I want to talk about web browsers, how they interpret the HTML you write, differences between browsers, what are standards and to paraphrase the old joke "if Standards are so great, why are there so many of them?"
Finally, I'll talk about the thought process behind HTML5 and CSS3, why they were introduced and what they hope to achieve.
